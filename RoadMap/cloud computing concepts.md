Cloud Computing Basics

1. Cloud Computing

Definition:
Cloud Computing means using computers, storage, and software through the internet instead of your own computer.

Why it is used:
It helps save money, access data from anywhere, and use services without buying expensive hardware.

Real-time Example:

* Using Google Drive to store files online.
* Using Salesforce in a web browser without installing it.

---

2. Service Models

IaaS (Infrastructure as a Service)

Definition:
The cloud provider gives virtual servers, storage, and networking. You manage the software.

Example:
A company rents a virtual server from AWS EC2 instead of buying a physical server.

---

PaaS (Platform as a Service)

Definition:
The cloud provider gives a platform to build and deploy applications. You only write the code.

Example:
A Salesforce developer creates an app on the Salesforce Platform without managing servers.

---

SaaS (Software as a Service)

Definition:
Ready-to-use software that runs in a web browser.

Example:
Using Salesforce CRM, Gmail, or Microsoft 365 without installing them.

---

3. Deployment Models

Public Cloud

Definition:
Cloud services are shared by many users over the internet.

Example:
Using Google Drive or AWS.

---

Private Cloud

Definition:
Cloud services are used by only one organization.

Example:
A bank uses its own private cloud to keep customer data secure.

---

Hybrid Cloud

Definition:
A combination of Public Cloud and Private Cloud.

Example:
A hospital stores patient records in a private cloud and uses a public cloud for its website.

---

4. Benefits of Cloud

* Cost Saving: No need to buy expensive servers.
* Access Anywhere: Use data from any location with the internet.
* Easy to Scale: Increase or decrease resources when needed.
* Automatic Updates: The cloud provider handles updates.
* Backup & Recovery: Data is backed up and can be restored easily.

Real-time Example:
A startup uses Salesforce CRM and Google Drive. Employees can work from home, share files, and access customer data anytime without maintaining their own servers.
